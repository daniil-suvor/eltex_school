простой текстовый редактор на основе curses

нет реализации изменения размеров окна
при первом запуске фиксирует размеры окна и использует их

нет возможности переноса строки при ее переполнении
есть возможность создания новой строки при нажатии ENTER

Управление:
    F2 - сохранить изменения
    F3 - выход
